## 简介
Ghost Domains: Unmanned Warfare Sim（GDUWS），一个模拟无人作战的战术层对抗小游戏。玩家布置无人单位后自动进行战斗，胜负由损失比例判定。

## 项目结构
- `code/`：核心代码
- `docs/`: 文档
- `readme.md`: 本文件

## 如何运行

Windows.

### 1. 环境准备

安装 JDK 17+ ：

```shell
winget install EclipseAdoptium.Temurin.17.JDK
```

安装完之后，需要重启 PowerShell.

### 2. 编译生成 exe

```shell
cd GDUWS/code
.\build.ps1
```

### 3. 点击 exe 游玩

`code/dist/` 为自包含发行包，双击 `GDUWS.exe` 启动，或：

```shell
.\dist\GDUWS.exe
```

## 如何游玩
选关，部署单位，开始游戏。开始游戏后战斗会自动进行。

### 部署阶段
部署的单位可以指派为打击单位，也可以指派为侦察单位。侦察单位会尝试发现目标并尽可能避战；打击单位会在侦察单位发现目标后进行打击。左键点击空地部署单位，左键点击已部署单位可以切换打击/侦察，右键点击已部署单位将其移除。

### 战斗阶段
战斗全自动进行，因为是无人单位（笑）。左键点击单位可以查看其攻击范围和移动路线，左键框选可以一次性查看多个单位。鼠标滚轮可以缩放。右键拖拽空地可以移动视角。

### 结算判定
当某一方损失 >90% 时，或者长时间双方无伤亡时，游戏结算。

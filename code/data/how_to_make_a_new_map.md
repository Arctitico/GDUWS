## 说明
该文档介绍如何新建一个地图以及关卡。

## .map 文件结构

`.map` 文件是纯文本文件（若注释含中文请使用 **GBK** 编码），格式如下：

```
# 可选注释行
<cols> <rows> <tileSize>
[terrain]
<地形第0行字符>
<地形第1行字符>
...
[decoration]
<装饰第0行字符>
<装饰第1行字符>
...
[deploy]
<部署第0行字符>
<部署第1行字符>
...
```

### 文件头

- **第1行（可选）：** 以 `#` 开头的注释。
- **第2行：** 三个整数 — `列数` `行数` `格子像素大小`（如 `40 24 20` 表示 40列、24行、每格20像素）。

### 分节

每个分节以标签行（`[terrain]`、`[decoration]`、`[deploy]`）开始，后跟恰好 `rows` 行、每行恰好 `cols` 个字符。

---

## 地形层 — `[terrain]`

定义每个格子的基础地形类型。

| 字符 | TerrainType | 通行性 | 含义 |
|------|-------------|--------|------|
| `.`  | GRASS       | LAND   | 草地（默认） |
| `,`  | DIRT        | LAND   | 泥土 |
| `s` / `S` | SAND   | LAND   | 沙地 |
| `#`  | MOUNTAIN    | BLOCK  | 山地（陆地和水上单位不可通行，仅空中单位可通行） |
| `_`  | SHALLOW     | WATER  | 浅水 |
| `~`  | WATER       | WATER  | 水 |
| `=`  | DEEP        | WATER  | 深水 |

未识别的字符默认为 GRASS。

## 装饰层 — `[decoration]`

定义地形之上的视觉装饰（纯视觉，不影响通行性）。

| 字符 | Decoration | 含义 |
|------|------------|------|
| `F`  | FLOWER     | 花朵 |
| `T`  | TREE       | 针叶树 |
| `t`  | TREE2      | 针叶树（变体） |
| `b` / `B` | BUSH  | 灌木 |
| `R` / `r` | ROCK  | 岩石 |
| `.`  | null       | 无装饰（默认） |

未识别的字符表示无装饰。

## 部署层 — `[deploy]`

定义玩家是否可在该格子部署单位。

| 字符 | 含义 |
|------|------|
| `X` / `x` | 禁止部署 |
| `.` 或其他 | 允许部署（默认） |
